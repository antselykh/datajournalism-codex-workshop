# Prompts for the workshop

Ниже — готовые prompts для работы с OpenAI Codex во время мастер-класса.

## 1. Первый prompt: понять, что за данные

```text
Посмотри файл births_regions_russia_2015_2020.csv в текущей папке.
```

Сделай 4 вещи:
1. Объясни простыми словами, что это за данные.
2. Назови все столбцы и что они означают.
3. Проверь, нет ли в файле пропусков, дубликатов и странных значений.
4. Предложи 5 журналистских вопросов, которые имеет смысл задать этому датасету.

## 2. Поиск аномалий

Изучи births_regions_russia_2015_2020.csv и найди 3 потенциально интересные аномалии или неожиданные паттерны.

Для каждой находки:
1. Объясни, что именно выглядит необычно.
2. Покажи, как ты это проверил.
3. Напиши, это похоже на реальную аномалию или может быть просто эффект масштаба.
4. Отдельно укажи, что журналист должен перепроверить вручную.

## 3. Построить график

Выбери самую интересную находку из предыдущего шага и построй один понятный график, который ее показывает.

Требования:
1. Используй Python.
2. Сохрани график как PNG.
3. Добавь понятный заголовок.
4. После построения коротко объясни, почему выбрал именно такой тип графика.

## 4. Короткая заметка по итогам

На основе найденных паттернов и построенного графика напиши короткую заметку для дата-журналиста в 5–7 предложений.

Структура:
1. Что видно в данных.
2. Почему это может быть интересно.
3. Что нужно перепроверить.
4. Какой следующий шаг расследования ты бы предложил.

## 5. Дополнительное упражнение: разбивка по полу

Посмотри файл births_regions_russia_by_sex_2015_2020.csv.

Найди регионы, где соотношение мальчиков и девочек среди родившихся выглядит необычно.

Для ответа:
1. Опиши, что именно кажется необычным.
2. Проверь, не связано ли это с маленькими абсолютными значениями.
3. Покажи 3 самых интересных кейса.
4. Предложи, что из этого может быть реальной историей, а что, скорее всего, статистический шум.

## 6. Если хочется копнуть глубже
Сравни несколько способов поиска аномалий в births_regions_russia_2015_2020.csv.

Например:
- абсолютное изменение
- относительное изменение
- сравнение с медианой по регионам
- поиск выбросов внутри временного ряда

Объясни, какой способ в этом датасете дает самые полезные результаты для журналиста и почему.

## 7. Если хочется получить идеи для материала

Предложи 5 возможных сюжетов для дата-журналистского материала на основе файла births_regions_russia_2015_2020.csv.

Для каждого сюжета:
1. Сформулируй гипотезу.
2. Объясни, почему она может быть важна.
3. Напиши, какие данные или комментарии нужны для проверки.
