# Codex for Data Journalists

Учебный репозиторий для мастер-класса по OpenAI Codex в GitHub Codespaces.

Задача занятия — показать, как ИИ-агент помогает дата-журналисту:
- быстро понять структуру таблицы
- найти аномалии
- построить первый график
- превратить находку в рабочую гипотезу

## Что в репозитории

- `births_regions_russia_2015_2020.csv` — основной датасет
- `births_regions_russia_by_sex_2015_2020.csv` — дополнительный датасет
- `.devcontainer/devcontainer.json` — настройка среды для Codespaces

## Что нужно

- аккаунт GitHub
- аккаунт ChatGPT с доступом к Codex
- Chrome или Edge

## Материалы занятия

- [Prompts for the workshop](./prompts.md)
- [Troubleshooting](./troubleshooting.md)

## Как начать

1. Откройте репозиторий в Codespaces.
2. Откройте терминал: `Terminal → New Terminal`
3. Запустите Codex:

```bash
codex
```
Если обычный вход не сработал, используйте:

```bash
codex login --device-auth
```

## Prompt для поиска аномалий
Изучи births_regions_russia_2015_2020.csv и найди 3 самые интересные аномалии или неожиданные паттерны.

Для каждой находки:
1. Коротко объясни, что выглядит необычно.
2. Коротко покажи, как ты это проверил.
3. Напиши, это похоже на реальную аномалию или может быть эффектом масштаба.
4. Укажи, что журналисту нужно перепроверить вручную.
5. Коротко свяжи находку с одним или несколькими из 5 журналистских вопросов, которые ты предложил раньше.

Отвечай коротко и структурно.

Формат ответа:
КОРОТКО:
- 3 находки
- по 1–2 строки на каждую

КАК ПРОВЕРЯЛ:
- максимум 3 коротких пункта на находку

ЧТО ПРОВЕРИТЬ ВРУЧНУЮ:
- максимум 2–3 коротких пункта на находку

ВОЗМОЖНЫЕ ПРОБЛЕМЫ С ДАННЫМИ:
- только если они есть

Не пиши длинное вступление и длинные абзацы.

## Prompt для графика
Выбери самую интересную находку из предыдущего шага.

Построй статичную choropleth-карту России по регионам в Python и сохрани как PNG.

Используй:
- GeoJSON: map_russia_2024.geojson
- CSV: births_regions_russia_2015_2020.csv

Требования:
- показывай только Россию, без соседних стран;
- не рисуй карту в сыром lon/lat, если это растягивает или сплющивает геометрию;
- выбери подходящую проекцию для России и коротко объясни выбор;
- сохрани корректный aspect ratio;
- если названия регионов в CSV и GeoJSON различаются, добавь NAME_FIXES;
- регионы без данных покрась в серый;
- сделай читаемый заголовок, легенду и аккуратные границы регионов;
- перед сохранением проверь, что карта не выглядит как растянутый глобус;
- работай в текущей папке проекта;
- если PNG не получается экспортировать корректно, честно объясни причину и сохрани HTML, а не делай кустарный fallback.

Отвечай коротко и структурно.

Формат ответа:
КОРОТКО:
- какую находку ты выбрал
- что показывает карта
- почему для карты выбран именно этот сюжет

КАК ПРОВЕРЯЛ:
- по каким полям связал CSV и GeoJSON
- какие регионы не сматчились, если такие есть

ЧТО ПРОВЕРИТЬ ВРУЧНУЮ:
- максимум 3 коротких пункта

ВОЗМОЖНЫЕ ПРОБЛЕМЫ С ДАННЫМИ:
- только если они есть

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
- какая проекция выбрана
- были ли NAME_FIXES
- PNG или HTML сохранен
- только коротко, без длинного отчета 

## Prompt для короткой заметки
На основе найденных паттернов и построенного графика напиши очень короткую заметку для дата-журналиста.

Структура:
1. Что видно в данных.
2. Почему это может быть интересно.
3. Что нужно перепроверить.
4. Какой следующий шаг расследования ты бы предложил.

Требование:
- 5–7 предложений максимум
- без длинных вводных
- без канцелярита
- без лишней уверенности

После заметки добавь отдельный блок:

ЧТО ПРОВЕРИТЬ ВРУЧНУЮ:
- максимум 3 коротких пункта

## Важно

Codex помогает ускорить разведку в данных, но не заменяет журналистскую проверку. Любой вывод нужно перепроверять по данным, методологии и контексту.

## Если что-то не работает

**Codex не запускается**

Проверьте, что вы открыли именно codespace, а не просто веб-редактор GitHub.

**Сломался браузерный логин**

Используйте:

codex login --device-auth

**Не видно терминала**

Откройте его через Terminal → New Terminal
или через палитру команд:

Cmd + Shift + P на Mac
Ctrl + Shift + P на Windows
